# Installation

## Target hardware

ByteBay was developed and tested on a **Kobol Helios64** (RK3399, 4× SATA). It should work on other ARM64 boards with Debian/Armbian, Docker, and `mdadm`.

## Requirements

| Component | Purpose |
|-----------|---------|
| Docker + Compose | `engine` and `web` containers |
| Go 1.22+ | Build `bytebay-agent` |
| mdadm | RAID |
| smartmontools | SMART |
| netplan.io | Network panel (netplan) |

## Automated install

```bash
git clone https://github.com/YOUR_USER/ByteBay.git
cd ByteBay
cp .env.example .env
```

Edit `.env`:

- `BYTEBAY_ADMIN_PASSWORD` — web panel password
- `BYTEBAY_SECRET_KEY` — JWT secret (generated by `install.sh` if missing)
- `BYTEBAY_DATA_PATH` — system data (SQLite DB, etc.)
- `BYTEBAY_VOLUMES_PATH` — NAS volume root mounted into engine

Then:

```bash
sudo ./deploy/install.sh
```

The script:

1. Installs missing dependencies
2. Creates the `bytebay` group and `/run/bytebay`
3. Builds and installs `bytebay-agent` (systemd)
4. Builds Docker images and runs `docker compose up -d`

## Accessing the panel

URL: `http://<ip>:8080` (port via `BYTEBAY_WEB_PORT`).

Credentials: `BYTEBAY_ADMIN_USER` / `BYTEBAY_ADMIN_PASSWORD` in `.env`.

## Upgrading

```bash
cd ByteBay
git pull
sudo ./deploy/install.sh
```

Or manually:

```bash
cd agent && go build -ldflags="-s -w" -o bytebay-agent ./cmd/bytebay-agent
sudo install -m755 bytebay-agent /usr/local/bin/
sudo systemctl restart bytebay-agent
docker compose build && docker compose up -d
```

## Important paths

| Path | Description |
|------|-------------|
| `/etc/bytebay/agent.env` | Agent env (optional) |
| `/etc/netplan/90-bytebay.yaml` | Network managed by panel |
| `/var/lib/bytebay/` | Agent state (mounts, etc.) |
| `.env` | Docker Compose secrets (**do not commit**) |

## Troubleshooting

- **Agent / Engine offline** (dashboard): check sockets under `/run/bytebay/` and Docker mount `/var/bytebay/sockets` (see architecture).
- **No web access after network change**: use serial or local SSH; fix netplan manually if needed.
